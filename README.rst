****
youd
****

generate a base32 random uuid.

============
Installation
============

.. code-block:: bash

    $ git clone https://github.com/t-mart/youd.git
    $ cd youd
    $ pip install --editable .

=====
Usage
=====

.. code-block:: bash

    $ youd
    NFZLD7S6ONFJVLPGULVJKJG2FU

    $ youd | pbcopy # place uuid in clipboard
